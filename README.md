# How to Run?

Before running the application, you need to add your Azure API key and endpoint to your environment variables:

```bash
export AZURE_API_KEY=<Your Azure API Key>
export AZURE_ENDPOINT=<Your Azure Endpoint>
```
After that, you can start the application with Docker Compose:
```bash
docker compose up
```
